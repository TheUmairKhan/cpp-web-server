Welcome to lebron2016x4 Web Server

This full-featured HTTP server was built entirely from scratch in modern C++ using a modular architecture and SOLID design principles. Key capabilities include:

  • Custom-built routing system with longest-prefix matching and dynamic handler dispatching
  • Support for multiple request handlers, including:
    • EchoHandler - Reflects request data for debugging
    • StaticHandler - Serves static files like HTML, images, and text
    • CrudApiHandler - File-based JSON API for Create, Read, Update, Delete
    • MarkdownHandler - Parses and converts Markdown to HTML using libcmark
    • HealthHandler - Reports server health status
    • SleepHandler - Simulates delayed responses (e.g., sleep 5 seconds)
    • NotFoundHandler - Custom 404 handler for unmatched routes
  • Markdown-to-HTML support using libcmark, dynamically linked and validated via CMake
  • Containerized with Docker - Multi-stage build for test + deployment with isolated dependencies
  • Hosted on Google Cloud Platform with VM-based deployment and firewall configuration
  • CI/CD Ready - Integrated with Cloud Build for seamless updates
  • Thoroughly tested with GoogleTest for C++ unit tests and Python for integration testing
  • Comprehensive logging using Boost.Log for debugging and observability

This endpoint supports file operations using HTTP methods. Interact using curl, Postman, or custom clients:

  • GET /api/<filename> - Retrieve file contents
  • POST /api/<filename> - Create a new file
  • PUT /api/<filename> - Overwrite an existing file
  • DELETE /api/<filename> - Delete a file
